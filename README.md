# Word Collector Game

A fast-paced word game where letters fall from the top of the screen and you race to collect them and spell valid words before they disappear. Built as a React web app.

## Play

Live at [watts4.github.io/wcg](https://watts4.github.io/wcg/)

## How to Play

- Letters fall from the top of the screen
- Click or tap letters to collect them and build a word
- Submit before the letters hit the bottom
- Only valid English words score points
- Difficulty increases over time — letters fall faster

## Features

- Full English dictionary validation (checks against a wordlist)
- Local high score tracking
- Difficulty scaling with adjustable fall speed
- Drag-to-collect letter mechanics
- Player name entry for leaderboard

## Stack

- React + Vite
- Deployed to GitHub Pages
